Movie API Documentation

Welcome to the Movie API!
This API allows users to add, modify, and delete values in the user field,
and access information in the movie field such as title, director, or genre.

Endpoints

Movies

Get All Movies

  • URL: /movies
  • Method: GET
  • Description: Get a list of all movies.
  • Response:
    • 200 OK: Returns an array of movie objects.
    • 400 Bad Request: If there are no movies in the database.

Get Movie by Title

  • URL: /movies/:title
  • Method: GET
  • Description: Get details of a specific movie by its title.
  • Request Parameters:
    • title: Title of the movie.
  • Response:
    • 200 OK: Returns details of the movie.
    • 400 Bad Request: If the movie with the specified title does not exist.

Get Movies by Genre

  • URL: /movies/genre/:genreName
  • Method: GET
  • Description: Get movies belonging to a specific genre.
  • Request Parameters:
    • genreName: Name of the genre.
  • Response:
    • 200 OK: Returns an array of movies of the specified genre.
    • 400 Bad Request: If the specified genre does not exist in the database.

Get Movies by Director

  • URL: /movies/director/:directorName
  • Method: GET
  • Description: Get movies directed by a specific director.
  • Request Parameters:
    • directorName: Name of the director.
  • Response:
    • 200 OK: Returns an array of movies directed by the specified director.
    • 400 Bad Request: If the specified director does not exist in the database.

Users

Add User

  • URL: /users
  • Method: POST
  • Description: Add a new user.
  • Request Body:
    • name: Name of the user.
  • Response:
    • 201 Created: Returns details of the newly created user.
    • 400 Bad Request: If the request body is missing the name field.

Add Favorite Movie to User

  • URL: /users/:id/:movieTitle
  • Method: POST
  • Description: Add a favorite movie to a user's list.
  • Request Parameters:
    • id: ID of the user.
    • movieTitle: Title of the movie to be added to favorites.
  • Response:
    • 200 OK: Indicates successful addition of the movie to the user's favorites.
    • 400 Bad Request: If the specified user does not exist.

Remove Favorite Movie from User

  • URL: /users/:id/:movieTitle
  • Method: DELETE
  • Description: Remove a favorite movie from a user's list.
  • Request Parameters:
    • id: ID of the user.
    • movieTitle: Title of the movie to be removed from favorites.
  • Response:
    • 200 OK: Indicates successful removal of the movie from the user's favorites.
    • 400 Bad Request: If the specified user does not exist.

Delete User

  • URL: /users/:id
  • Method: DELETE
  • Description: Delete a user.
  • Request Parameters:
    • id: ID of the user to be deleted.
  • Response:
    • 200 OK: Indicates successful deletion of the user.
    • 400 Bad Request: If the specified user does not exist.

Update User

  • URL: /users/:id
  • Method: PUT
  • Description: Update user information.
  • Request Parameters:
    • id: ID of the user to be updated.
  • Request Body:
    • name: Updated name of the user.
  • Response:
    • 200 OK: Returns details of the updated user.
    • 400 Bad Request: If the specified user does not exist.

Error Handling

  • 500 Internal Server Error: Indicates an unexpected error occurred on the server.
