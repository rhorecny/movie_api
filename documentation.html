Documentation for movie_api

movie_api will be built as an API to return a list of all movies to a user including genre, description, image URL, and whether it is a feature film or not. There will be an ability to register new users, update info (username, password, date of birth, email) allow users the ability to create their own lists to their favorites, allow users to remove from their favorites, and allow existing users to deregister. This API will be built using node.js using REST achitecture.
